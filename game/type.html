游戏

游戏分类

游戏主要分为两种：自动寻路、自寻死路

按运行平台划分：

  • 手游(手机)
  • 页游(浏览器)、
  • 端游(PC客户端)、
  • TV GAME
    • 家用主机(PS4、Xbox、Switch)
    • 掌机(Switch)

TV GAME ：“电视游戏”在这里的范围包含“游戏机”和“游戏软件”。这个名词是从日文（テレビゲーム）翻译而来，是日本独创的和制英语，又称作“游戏机游戏”，英文为“Video Game”或“Console game”。

游戏类型：

ACT、FTG、STG、FPS、SLG、RTS、RTT、RPG、AVG、SIM、SPG、RAC、PUZ

、MUG、LVG、CAG、ETC

RPG：Role-playing game角色扮演游戏

A.RPG：Action Role Playing Game动作角色扮演类游戏

SRPG：Strategy Role-Playing Game策略角色扮演游戏

ACT：Action Game动作类游戏

FTG：Fight Technology Game 格斗类游戏

LVG：Love Game 恋爱养成游戏

PZL：Puzzle Game 解谜类游戏

RAC：Race Game 竞速类游戏

AVG：Adventure Game冒险类游戏

A.AVG：ActionAdventure Game 动作冒险游戏

SPG：Sport Game 体育类游戏

MUG：Music Game 音乐类游戏

PUZ：Puzzle Game 益智类游戏

SIM/SLG（模拟经营游戏）

：Simulation Game （模拟养成游戏）

：Education Simulation （策略类游戏）

：Strategy Game

RTT：Real-Time Tactics 即时战术游戏

RTS：Real-Time Strategy 即时战略游戏

STG：Shooting Game 射击类游戏

FPS：First-Person Shooter Game 第一人称射击类游戏

TAB：Table桌面类游戏
