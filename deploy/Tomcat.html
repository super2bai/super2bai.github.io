Tomcat

同一个Tomcat多端口多项目部署

背景

当服务器只有一台，又需要将不同环境分开管理，可以选择多Tomcat分别部署不同的项目，也可以选择同一个Tomcat下多端口多项目部署。

操作

修改配置文件

配置文件位置：$TOMCAT_HOME/conf/server.xml

需要修改的地方
  • Service 标签 的 name 属性
  • 端口号( Connector 标签的 port 属性)
  • Engine 标签的 name 属性
  • Host 标签的 appBase 属性
  • 注释掉 port为8009的Connector

复制文件夹

文件夹位置：$TOMCAT_HOME/webapps

  • 复制此文件夹
  • 修改名称分别为上方配置文件中Host节点中appBase中的值
  • 把不同项目的war包分别放到两个文件夹下

重启Tomcat

  • 观察日志是否有错误
  • 测试每一个项目是否能正常访问
  • 测试每一个项目是否能正常提供数据

完毕
