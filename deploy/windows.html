Java Web部署到Windows的Tomcat

关键字

  • Java Web
  • Windows
  • Maven
  • Tomcat
  • 部署

下载

  • JDK
  • Tomcat

环境变量

JDK

  • JAVA_HOME：根文件夹
  • JRE_HOME：JAVA_HOME\jre

Tomcat

  • TOMCAT_HOME：根文件夹\bin

部署服务

  • eclipse->Export->WAR file
  • 将war包放在Tomcat下的webapps文件夹内

后台运行

  • 运行Tomcat目录的bin文件夹的service文件
    • 我的电脑->管理->服务和应用程序->服务->Apache Tomcat 8.5 tomcat
  • 启动

问题总结

访问tomcat出现java.lang.IllegalStateException No output folder错误解决方法

Tomcat文件夹->右键->安全->编辑->Users->完全控制->允许->打勾

Tomcat的localhost:8080拒绝访问

描述：运行Tomcat后访问localhost:8080，访问不到。

解决：控制面板->程序->启用或关闭Windows功能->Internet Information Service->勾选
