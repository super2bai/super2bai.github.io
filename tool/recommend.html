工具推荐

一些我自己用起来很爽的工具。

Mac

  • 快速启动：Alfred
  • 终端：oh-my-zsh+ iTerm
  • MySQL：Sequel Pro
  • 壁纸：pap.er

  • 键位映射：Karabiner Elements

  • 绘图：OmniGraffle

  • OpenVPN：Tunnelblick

  • 窗口管理：ShiftIt

  • 快速启动和切换APP：Manico

  • 防休眠，演讲不尴尬：Caffeine

  • 截图识字：iText

Windows

  • 快速启动：Wox

多平台

  • 局域网传输文件：
    • Dokto
    • 把需要传送的目录移动到新建的目录中，然后在当前目录输入python3 -m http.server
  • Markdown编辑器：Typora

  • 远程协作：TeamViewer

  • 绘图：draw.io
